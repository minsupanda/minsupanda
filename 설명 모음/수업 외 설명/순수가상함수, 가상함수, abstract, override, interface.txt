[가상함수]
이 함수를 오버라이딩(재정의) 할 수도 있고 안 할수도 있다

[순수가상함수]
이 함수를 반드시 오버라이딩 할 것이다!

['abstract' == '=0']
순수가상함수 선언하는 것

[override]
자식 Class에서 이 함수는 순수가상함수로 만들어진 것이다!

	__interface Scene // 순수가상함수만 가진 애들
	// __interface 얘로 선언한다는 것이 이런 기능이 있다고 알려주는 것이기 때문에
	// 이 interface를 상속받은 애들은 반드시 public으로 만들어줘야 하고 구현을 해줘야 한다
	// interface 상속받은 클래스들을 선언한 interface*로 하나로 관리하면서 upacasting으로 사용할 수 있다
	// interface 상속 받은 클래스는 반드시 정의해줘야 한다.
	// 추상클래스는 변수를 만들 수 있지만, interface는 불가능하다.
	// c++를 제외한 다른 언어에서는 다중상속이 불가능한데 interface를 여러개 상속받아 다중상속을 할 수 있다.
