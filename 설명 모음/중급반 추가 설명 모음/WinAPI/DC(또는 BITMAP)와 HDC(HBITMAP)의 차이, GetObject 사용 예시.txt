[DC - HDC / BITMAP - HBITMAP의 차이]

* DC / BITMAP 자료형
- 각 자료형에 지정된 크기만큼 메모리를 차지하고 있다.
- DC tmp, BITMAP tmp : 하나의 오브젝트

* HDC, HBITMAP 자료형
- HANDLE값 : 메모리 주소가 정수로 치환된 형태
- 다시 말해 시작주소를 가지고 있다.
- 시작주소(핸들값)를 가지고 WinAPI에게 요청해서 저장된 데이터를 사용하는 방식
  -> 직접 데이터에 접근하지 못하기 때문에 핸들값을 가지고 요청하는 것이다.

---------------------------

[GetObject(HANDLE h, int c, LPVOID pv)]

- 사용 예시(카드게임 Bitmap 클래스 cpp파일 Init함수)
 GetObject(m_Bitmap, sizeof(BitMap_Info), &BitMap_Info);

- m_Bitmap은 HBITMAP이기 때문에 핸들값을 가지고 있다.
- BitMap_Info는 BITMAP 자료형으로 이미지 데이터가 필요하다.
- 그렇기때문에 HBITMAP(핸들값 = 시작주소)부터 시작해 BITMAP 자료형의 사이즈만큼 불러오는 것이다
