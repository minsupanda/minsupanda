**사용예제코드 같이보기**
[팩토리 메소드]
- 공장처럼 하나의 객체를 찍어내는 것
ex. 게임에서 오크들이 생성될 때 함수를 호출해 오크를 만드는 것

(예제코드)
캐릭터를 찍어낼 공장 클래스를 상속받은 직업 생성 클래스에서 직업을 생성하면공장에서 오버라이딩된 해당 직업에 맞는
Create 함수를 호출해 private영역(초기화 과정을 숨기고 반환타입을 제어?)에 있는 공장함수에서 해당 직업에 맞는 캐릭터를 만들어낸다
-> 굳이 private로 숨기지 않아도 된다.

[추상 팩토리]
- 하나의 클래스(공장)으로 여러개의 객체를 만들어 주는 것
ex. 아쳐라는 클래스를 만들어낼 때 이에 더해 장비를 포함한 여러가지 부가적인 객체들을 추가해서 만들어내는 것

* 굳이 이 둘을 확연하게 구분해두지 않는다
-> 팩토리 메소드의 여러 묶음을 추상팩토리라고 생각해도 큰 상관은 없다? -> ㅇ
